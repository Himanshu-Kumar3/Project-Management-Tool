## Installation guidlines
 ## Mongodb and Node js must be install

 - Mongodb link:https://www.mongodb.com/try/download/community

 - Nodejs link: https://nodejs.org/en

 - project is divided in two folders 'client' and 'server'

 - download or clone the repo
 - open the folder in vs code
 - split the terminal in two parts
 - In first terminal type 'cd client' and type'npm install' and in second terminal type'cd server' and type 'npm install'
 - if 'uploads' folder is not present in the 'server' folder please create a folder name 'uploads' inside the 'server' folder
 - now type 'npm run dev' in both the terminals'
 - Now go to any browser and type 'localhost:5173'
